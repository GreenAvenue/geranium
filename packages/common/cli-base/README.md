## Goの環境構築とプロジェクト作成手順 (TODO: あとでwikiリポジトリでまとめる)

- mac

```bash
# go のインストールと確認
brew install go
which go

# 以下の`GO111MODULEをonにする`コマンドを実行するかログインシェルなどに記載しておく
export GO111MODULE=on

# go のパッケージ管理ファイルの初期化
go mod init
```
